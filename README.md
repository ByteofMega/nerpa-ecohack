# Nerpa — детекция морского мусора (ЭкоХак: Чистый Океан)

Прототип системы для поиска и подтверждения мусорных пятен в океане:
спутниковый снимок находит подозрительную зону, автономный подводный
аппарат (АНПА) подплывает к точке и подтверждает наличие мусора
локальной съёмкой, после чего строится тепловая карта загрязнения.

Сделан за 24 часа на хакатоне "ЭкоХак: Чистый Океан" (март 2026,
Казань, ИТ-парк).

## Как это работает

Спутниковое фото → YOLO (спутник) → боксы: мусор / водоросли / пятно
→ АНПА плывёт к точке (имитация GPS) → кадр с дрона/АНПА → YOLO
(подводная) → боксы: мусор / водоросли → тепловая карта (overlay по
всем боксам).

## Важное честное примечание

Во время хакатона мы обучали и тестировали YOLO-модели отдельно на
реальных изображениях (спутниковые снимки и подводные кадры) и
получили рабочую детекцию мусора. Но за 24 часа не успели полностью
интегрировать веса моделей в интерфейс — поэтому в этой версии, если
файлы весов (weights/satellite_best.pt, weights/drone_best.pt) не
найдены, приложение автоматически переключается в демо-режим и
показывает искусственные (заранее заданные) боксы, чтобы можно было
показать логику работы системы целиком.

Если положить обученные веса в папку weights/, приложение
использует реальные модели через models.py (ModelManager).

## Стек

- Python, Tkinter — интерфейс
- OpenCV, PIL — обработка изображений и вывод в UI
- Ultralytics YOLOv8 — детекция объектов
- NumPy — генерация тепловой карты (Gaussian blur по боксам)

## Структура

- main.py — интерфейс, сценарий миссии
- models.py — ModelManager, загрузка YOLO весов и инференс
- drone.py — имитация движения АНПА к точке (GPS)
- logic.py — сверка результатов двух моделей
- utils.py — геометрия, отрисовка боксов, тепловая карта
- weights/ — сюда кладутся satellite_best.pt и drone_best.pt

## Запуск
```python
pip install ultralytics opencv-python pillow numpy
python main.py
```

## Идея

Мусорные пятна в океане сложно обнаружить и подтвердить только
спутником — разрешение снимков не всегда позволяет отличить мусор от
водорослей или естественной ряби. Связка "спутник + автономный
подводный аппарат" даёт двойное подтверждение: сначала находим
подозрительную зону сверху, потом подтверждаем её на месте.
